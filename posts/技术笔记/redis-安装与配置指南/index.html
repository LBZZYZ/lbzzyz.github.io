Redis 安装与配置指南

March 8, 2021

Ubuntu安装

sudo apt update
sudo apt install redis-server

实际会有三个instance被安装：

  1. libjemalloc1
  2. redis-server
  3. redis-tools

检查安装是否成功

$ redis-cli
127.0.0.1:6379>

输入ping,有下方回显说明连接成功

127.0.0.1:6379> ping
PONG
