eslint-plugin-vue

Vue.js 官方 [[ESLint]] 代码检查插件。

检查范围：

  • .vue 后缀文件中的 <templete> 和 <script> 标签。
  • .js 文件中的 vue 代码。

检查内容：

  • 语法错误
  • 违反 Vue 官方风格指导的写法
  • 错误使用 官方内置指令

命令行检查：

npx eslint src

https://eslint.vuejs.org/ https://github.com/vuejs/eslint-plugin-vue
